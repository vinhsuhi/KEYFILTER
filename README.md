# KEYFILTER - VinhTong

## INPUT

Store data in 'data.txt'

## OUTPUT

Output will be store in 'output.txt'

## Dependencies
* Python 3.x (tested on Python 3.6)
* pytorch
* sentence-transformers
* regex

To install all the requirements, run the following comman: 

```bash
pip install -r requirements.txt
```

## Usage

Run

```bash
python keyfilter.py
```
